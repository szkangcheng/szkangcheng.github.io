apache中的.htaccess绕过

.htaccess 是 Apache HTTP 服务器中的一个配置文件，用于在目录级别上管理服务器行为。它允许用户在不修改主服务器配置文件的情况下，覆盖全局配置设置。常见的用途包括 URL 重写、访问控制、自定义错误页面等。

主要功能

  1. URL 重写：通过 mod_rewrite 模块实现 URL 重定向和重写。
  2. 访问控制：限制特定 IP 或用户的访问。
  3. 自定义错误页面：定义 404、500 等错误页面。
  4. MIME 类型设置：指定文件类型的处理方式。
  5. 缓存控制：设置浏览器缓存策略。
